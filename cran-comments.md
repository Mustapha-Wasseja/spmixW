## Package description

First submission. spmixW implements Bayesian spatial panel data models with
convex combinations of spatial weight matrices, following the methodology of
Debarsy and LeSage (2021, Journal of Business & Economic Statistics). The
package provides MCMC estimation for SAR, SDM, SEM, SDEM, OLS, and SLX panel
models, with support for multiple weight matrices, Bayesian model averaging,
and model comparison via log-marginal likelihoods. No existing CRAN package
provides this functionality.
